maximo7.6安装教程

maximo
  • maximo7.6

Maximo7.6安装教程，包括DB2、WebSphere
英文原文地址

解压安装文件

  • 将下载的安装文件多个部分解压到一个单个文件夹中 下载地址

执行launchpad64.exe.

基础配置

  • 勾选(DB2 和 WebSphere) 然后点击 Install IBM Maximo Asset Management
  • 勾选所有的安装包，点击下一步
  • 选择同意license

配置安装目录

  • 可保持默认直接点击下一步
  • 可选择相应的组件更改目录 也可以保持默认

语言选择

选择所有功能

DB2配置

IBM HTTP Server配置

安装

  • 点击install
  • 选中 IBM Tivoli process automation engine configuration utility option 点击完成.

配置WebSphere

  • 点击 Prepare WebSphere Application Server for configuration .
  • 如果安装时没有指定一下信息则保持默认，如果指定则按执行修改，点击下一步
  • 保持默认，然后输入管理密码，此处需记录管理密码，点击下一步
  • 保持默认点击下一步
  • 选择默认授权方式
  • 勾选所有选项,点击下一步
  • 等待配置过程

安装Maximo7.6

  • 运行安装目录中的launchpad64.exe.选中 IBM Maximo Asset Management v7.6 然后点击 Install IBM Maximo Asset Management components
  • 选中安装包，点击下一步
  • 同意license
  • 配置已配置的目录点击下一步
  • 选中IBM Maximo Asset Management 7.6.0.0
  • 点击 install

配置maximo7.6

  • 打开工具,点击 configure a new deployment
  • 配置DB2、WebSphere及默认用户密码
  • 配置主题，可以选择勾选创建一个样例数据到数据库中
  • 配置DB2 instance
  • 配置DB2 数据库信息
  • 配置HTTP Server信息
  • 配置maximo用户信息
  • 选择基础语言
  • 选择支持的其他语言
  • 选择所有的部署选项，点击finish
  • 安装过程

登录maximo
